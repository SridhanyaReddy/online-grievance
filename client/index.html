How Grievance Hub Works

Transparent, step-by-step redressing workflow for public peace of mind.

1. File Complaint

Submit details, pin maps, and attach site images.

2. verification

System sanitizes input and automatically predicts department.

3. Resolve

Managers assign issues to field officers for redressal.

4. Close Case

Download receipt and rate redressal efficiency.
